# project_x

A currently unnamed GPL3 open-world survival roguelikelike game written in C++

#### Try it
    $ sudo apt-get install liballegro5-dev liballegro-image5-dev
    $ git clone https://github.com/dvolk/project_x
    $ cd project_x/src
    $ make
    $ cd ..
    $ ./project_x
